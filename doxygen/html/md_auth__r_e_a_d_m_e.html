EdDSA Flavour

  • key exchange: NIST SP 800-186 Elliptic Curve Diffie-Hellman (ECDH) X25519.
  • signing/verification: NIST FIPS PUB 186-5 Ed25519.
  • MAC: RFC7539 Poly1305.
  • encryption: XSalsa20 stream cipher.
  • base lib: Sodium v1.0.18
  • pros:
    • IoT friendly.
    • IOTA 1.5 compatible.
    • cryptographically robust.
    • EdDSA is younger than RSA.
  • cons:
    • unknown vulnerabilities != no vulnerabilities.
  • usage scenarios: Edge Infrastructure and POSIX compliant RTOSs.

RSA Flavour

  • key exchange: NIST FIPS PUB 800-56A Diffie-Hellman (DH).
  • signing/verification: NIST FIPS PUB 800-131A Rivest-Shamir-Adleman (RSA) 2048+ bits.
  • HMAC: NIST FIPS PUB 198-1 SHA-256
  • encryption: NIST FIPS PUB 197 AES256
  • base lib: OpenSSL v1.0.2
  • pros:
    • OpenSSL friendly.
    • RSA is still the leading DSA for most certificate applications.
  • cons:
    • hungry for CPU+RAM (not IoT friendly).
    • RSA known vulnerabilities, especially Shor.
  • usage scenarios: Cloud Infrastructure and Embedded Linux.
